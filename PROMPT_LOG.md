## Исправление ошибок в процессе работы

### Проблема 1: Путь к бинарнику в Python
**Ошибка:** Python-скрипты не находили http-server.exe
**Решение:** Использовал абсолютные пути E:/laba9/go-task2/http-server.exe

### Проблема 2: Тесты TCP-сервера
**Ошибка:** Тесты падали, потому что сервер не успевал запуститься
**Решение:** Добавил time.Sleep(100 * time.Millisecond) перед подключением

### Проблема 3: Бинарник зависает в subprocess
**Ошибка:** HTTP-сервер не завершался после одного запроса
**Решение:** Добавил проверку аргументов командной строки для двух режимов

### Проблема 4: Rust не установлен
**Ошибка:** maturin failed - rustc not found
**Решение:** Установил Rust через rustup.rs

### Проблема 5: Таймауты при установке пакетов
**Ошибка:** ReadTimeoutError при pip install
**Решение:** Использовал зеркало Tsinghua https://pypi.tuna.tsinghua.edu.cn/simple

### Проблема 6: Maturin требует виртуальное окружение
**Ошибка:** Couldn't find a virtualenv or conda environment
**Решение:** Создал виртуальное окружение через py -m venv venv
